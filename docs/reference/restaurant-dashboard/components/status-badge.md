# Component: Status Badge / Pill

A small coloured label conveying state. The colour vocabulary is consistent product-wide. **Evidence:** Observed.

## Colour → meaning

| Colour | Tokens seen | Meaning |
|---|---|---|
| 🟡 Gold | "VACANT", active filter tab | free / neutral-active |
| 🔴 Red | "ALLOTED", "alloted", delete | occupied / danger |
| 🟢 Green | "Active", "Returning Guest", "Reboot", 👍 | positive / online |
| 🟣 Purple | "Reserved" | held by reservation |
| 🔵 Blue | "Shutdown" | info / neutral action |

## Forms

- **Pill** — filled rounded label on cards (table status, reservation status, via/source).
- **Dot** — small coloured circle (Transfer Sessions grid).
- **Count chip** — e.g. "9 visits" (Guest List), tab counts "Active (16)".
- **Request badge** — dark chip surfacing an active table request flag.

## Guidance

Reuse the same colour→meaning map for any new state so operators can read status by colour without reading text.
